# 风电机组新型智能消防系统仿真实验源码与流程

## 1. 文件说明

本文件夹提供课程设计报告中“算法比对及关键实验结论”部分对应的完整仿真实验源码。

| 文件 | 作用 |
|---|---|
| `config.py` | 实验参数、样本长度、类别名称、报警阈值 |
| `data_generator.py` | 生成正常、过热、阴燃、明火四类风机机舱仿真数据 |
| `preprocessing.py` | 平滑滤波、Min-Max归一化 |
| `features.py` | 提取统计特征、标准化二维特征 |
| `models.py` | 阈值法、BP网络、SVM、Numpy版LSTM模型 |
| `fuzzy_system.py` | LSTM输出与传感器隶属度的模糊融合 |
| `evaluate.py` | 准确率、误报率、漏报率、报警提前量等评价指标 |
| `plot_results.py` | 生成算法对比柱状图 |
| `run_all.py` | 一键运行全部实验 |
| `requirements.txt` | 依赖库列表 |

## 2. 实验目的

验证多传感器融合和LSTM-模糊推理方法在风电机组火灾早期识别中的有效性，并与固定阈值法、BP神经网络、SVM和单独LSTM方法进行对比。

## 3. 实验环境

建议环境：

- Python 3.9及以上
- `numpy`
- `Pillow`

安装依赖：

```powershell
pip install -r requirements.txt
```

如果使用本机 Codex 自带 Python，也可以直接运行源码；当前源码没有依赖 TensorFlow、PyTorch、scikit-learn 等重型库。

## 4. 具体实验流程

### 步骤1：生成仿真样本

运行 `run_all.py` 后，程序首先调用 `data_generator.py` 生成四类样本：

1. 正常运行：温度、烟雾、气体、电流波动均较低；
2. 部件过热：温度和红外热点持续上升；
3. 阴燃早期：烟雾和气体浓度先于温度明显上升；
4. 明火发展：温度、红外热点、烟雾和气体快速升高。

每类样本默认300组，共1200组；每组样本长度为60秒，采样间隔为1秒。

### 步骤2：数据预处理

程序对原始数据执行：

1. 三点滑动平均，减小传感器瞬时噪声；
2. Min-Max归一化，将不同量纲数据统一到0到1范围；
3. 按7:1:2划分训练集、验证集和测试集。

归一化后的数据会保存到：

```text
outputs/simulated_wind_turbine_fire_dataset.npz
```

### 步骤3：训练和运行对比模型

程序依次运行五种方法：

1. 固定阈值法：根据温度、烟雾、气体、红外热点和电流波动加权求风险；
2. BP网络：对统计特征进行非线性分类；
3. SVM：使用线性间隔分类思想区分正常与火灾风险；
4. LSTM：用Numpy实现门控时序网络，学习连续60秒的变化趋势；
5. LSTM+模糊推理：将LSTM概率与温度、温升速率、烟雾、气体等隶属度融合，输出综合风险指数。

### 步骤4：计算评价指标

评价指标包括：

| 指标 | 含义 |
|---|---|
| 准确率 | 全部样本中判断正确的比例 |
| 误报率 | 正常样本被误判为火灾风险的比例 |
| 漏报率 | 火灾风险样本被误判为正常的比例 |
| 平均报警提前量 | 系统首次报警时刻距离设定危险时刻的平均提前秒数 |
| 平均响应时间 | 系统在测试样本中的平均首次报警时刻 |

### 步骤5：输出实验结果

运行结束后生成：

```text
outputs/metrics.csv
outputs/algorithm_comparison.png
outputs/simulated_wind_turbine_fire_dataset.npz
```

其中 `metrics.csv` 可直接复制到课程设计报告中；`algorithm_comparison.png` 可作为算法比对图。

## 6. 结果解释建议

实验结果一般应体现以下规律：

1. 阈值法实现简单，但容易受噪声和环境波动影响，误报率较高；
2. BP和SVM能利用多变量统计特征，准确率较阈值法提高；
3. LSTM能够学习温升、烟雾和气体浓度的连续变化趋势，漏报率进一步降低；
4. LSTM+模糊推理兼顾模型预测与工程规则，可解释性更好，综合指标通常最优。

## 7. 注意事项

本实验数据为课程设计仿真数据，不是真实风场采集数据。若后续用于工程研究，应使用真实风机机舱运行数据重新训练模型，并根据具体风机型号、通风结构、传感器安装位置和灭火装置参数重新校准阈值。
